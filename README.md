# persistence
A Bus Schedule that keeps tracks of schedules and displays the next arrival time of each bus.
The Schedule is save in a firebase database. 

## Live Link
https://coolcat-310.github.io/persistence/


## Technology Used

- HTML 5
- CSS/Bootstrap 3
- jQuery 3.1
- firebase
